# 简单介绍
基于 cocos2dx 最新版的自研编辑器。经历数年的使用与迭代.

# 主要功能:
> 1.UI编辑.

> 2.基于 cocos action 机制的 动画编辑.

# 特色
> 1.编辑器编辑 UI 十分方便, 增加了很多提升开发游戏效率的节点以及属性.

> 2.UI组件化, 可以被其他UI复用.

> 3.UI节点类型多样, 扩展类型方便.

> 4.动画编辑器支持所有 cocos2dx 的 action 以及提供的方便的扩展机制. 能够及时地预览编辑出来的效果.

> 6.支持多国语言版本的开发.

> 7.强大的分辨率适配机制. 适配不同手机的分辨率不需要写代码. (并且分辨率适配机制也是可以扩展的)

> 8.编辑器实现了自己编辑自己的能力.
